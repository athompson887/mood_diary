# Support for Mood Diary

Thank you for using Mood Diary! We're here to help you track your emotional wellbeing and gain insights into your mood patterns.

## Getting Help

### Documentation

- **App Features**: Learn about mood logging, statistics, history view, and more within the app
- **Getting Started**: Launch the app and start logging your first mood entry
- **Privacy**: Read our [Privacy Policy](PRIVACY.md) to understand how your data is protected

### Report a Bug

If you've found a bug or issue:

1. Check [existing issues](https://github.com/athompson887/mood-diary/issues) to see if it's already reported
2. If not, [create a new issue](https://github.com/athompson887/mood-diary/issues/new) with:
   - Clear description of the problem
   - Steps to reproduce
   - Expected vs actual behavior
   - iOS/Android version and device model
   - Screenshots if applicable

### Feature Requests

Have an idea to improve Mood Diary?

1. Browse [existing feature requests](https://github.com/athompson887/mood-diary/issues?q=is%3Aissue+label%3Aenhancement)
2. [Submit a new feature request](https://github.com/athompson887/mood-diary/issues/new) describing:
   - What you'd like to see added
   - Why it would be helpful
   - How you envision it working

### Questions

For general questions about the app:

- Check the in-app settings and help options
- Browse [GitHub Discussions](https://github.com/athompson887/mood-diary/discussions) (if enabled)
- [Open an issue](https://github.com/athompson887/mood-diary/issues/new) with your question

## Technical Support

### System Requirements

- **iOS**: 13.0 or later
- **Android**: 8.0 or later
- **Devices**: iPhone, iPad, Android phones and tablets

### Common Issues

**Q: My mood entries aren't saving**
- Ensure the app has storage permissions
- Try restarting the app
- If the issue persists, report it via GitHub Issues

**Q: I lost my data after updating**
- Data is stored locally and should persist through updates
- If you have device backups enabled, restore from backup

**Q: Statistics aren't showing correctly**
- Ensure you have at least a few mood entries logged
- Try pulling down to refresh the statistics screen

**Q: The app crashes on launch**
- Try restarting your device
- Ensure you're running the latest version
- If the issue persists, report it via GitHub Issues

## Response Time

This is an independently maintained app. We aim to respond to issues within:

- **Critical bugs**: 1-3 days
- **Feature requests**: 1-2 weeks
- **General questions**: 3-5 days

Response times may vary based on complexity and availability.
